💱 Currency Converter (INR to Foreign Currency)

A simple Python-based Currency Converter that converts an amount in Indian Rupees (INR) to a selected foreign currency using exchange rates stored in a text file.

⸻

🚀 Features

* Convert INR to multiple currencies
* Reads exchange rates from a separate file (CurrencyData.txt)
* Dynamic currency list generation
* Beginner-friendly Python project
* Simple command-line interface

🛠️ How It Works

1. The program reads exchange rates from CurrencyData.txt.
2. Stores the data in a Python dictionary.
3. Takes an amount in INR from the user.
4. Displays all available currencies.
5. Converts the amount into the selected currency.
6. Displays the result.

📄 Sample CurrencyData.txt

USD	0.012
EUR	0.011
GBP	0.0095
JPY	1.73
AUD	0.018
